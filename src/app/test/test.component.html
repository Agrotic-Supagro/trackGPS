1. Using data-provide="datepicker" and data-date-show-on-focus="false" attributes means that the first time the datepicker-enabled input element receives focus, the calendar is displayed, which is incorrect. Thereafter, it is not displayed when the input receives focus.



  2. Using JS configuration ($.fn.datepicker.defaults.showOnFocus), the calendar is only displayed when the icon is clicked and not when the input receives focus, which is correct.
